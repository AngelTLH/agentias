Proyecto 01 · Visualización

Dashboard Comercial en Shiny de R

Panel interactivo de KPIs para ventas y priorización táctica.

Desarrollado como proyecto final del Curso de R. Integra carga automática de hojas locales de datos comerciales, limpiando registros e imputando nulos mediante lógica estadística avanzada.

Tecnologías usadas

R (Tidyverse, ggplot2, Shiny), Quarto y despliegue local responsivo.

Habilidades aplicadas

Limpieza y manipulación de variables, modelado descriptivo e interactividad visual en local.

Proyecto 02 · Predicción

AutoML de Demanda en Retail

Ajuste predictivo y backtesting local para reabastecimiento.

Desarrollado en el Bootcamp de Machine Learning. Entrena algoritmos supervisados de regresión en local para estimar el flujo de existencias en tiendas críticas.

Tecnologías usadas

Python (Pandas, Scikit-Learn, Optuna, Matplotlib).

Habilidades aplicadas

Optimización de hiperparámetros con Optuna, evaluación de curvas de aprendizaje y cálculo de residuos.

Proyecto 03 · Ingeniería MLOps

API de Riesgo en FastAPI y Docker

Clasificador de transacciones empaquetado como microservicio ligero.

Proyecto final del Máster en MLOps. Diseña y entrena un modelo clasificador XGBoost, exponiéndolo a través de una API REST estructurada en FastAPI y empaquetada en un contenedor Docker listo para producción.

Tecnologías usadas

Python, FastAPI, Pydantic, Docker, XGBoost, Uvicorn.

Habilidades aplicadas

Creación de endpoints seguros, validación de esquemas de datos, unit tests y contenedores aislados.

Proyecto 04 · Gobernanza

EDA Clínico y Anonimización Local

Pipeline de datos médicos enfocado en la privacidad de pacientes.

Desarrollado en el Módulo Estadístico. Un pipeline local para imputar nulos de datos clínicos de salud y aplicar técnicas de anonimización y encriptado ligero sobre variables de identidad.

Tecnologías usadas

Python (Pandas, Cryptography), SQLite local.

Habilidades aplicadas

Gobernanza ética de datos, anonimización por hash y análisis de correlación exploratoria.
