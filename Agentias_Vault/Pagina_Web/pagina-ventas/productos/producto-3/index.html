Máster en MLOps e Inteligencia Artificial

Aprende a productivizar modelos. Desarrollo de APIs en FastAPI, unit testing de modelos, empaquetado de contenedores Docker y despliegues en producción local u on-premise.

Detalles del Máster

El programa definitivo para convertirte en Ingeniero MLOps, uniendo ciencia de datos y operaciones en la nube.

  • Duración: 16 semanas (Clases avanzadas y proyecto de egreso).
  • Temario: APIs en FastAPI, empaquetado Docker y Kubernetes.
  • Automatización: CI/CD para modelos con GitHub Actions.
  • Privacidad: Implementación on-premise y gobierno ético.
  • Soporte comercial: Mentoría de carrera y preparación de entrevistas.
