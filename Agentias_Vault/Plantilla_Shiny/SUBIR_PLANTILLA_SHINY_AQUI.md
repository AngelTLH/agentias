# Subir Plantilla de R Shiny Aquí

Sube en esta carpeta tu plantilla personalizada de R Shiny (por ejemplo, el archivo `app.R` o los archivos `ui.R` / `server.R` y cualquier recurso asociado).

El sistema copiará automáticamente todo el contenido de esta carpeta al directorio `/Dashboard/` del proyecto cuando selecciones "Shiny (R)" como tecnología de dashboard.
