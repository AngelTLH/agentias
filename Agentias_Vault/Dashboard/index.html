¿Cómo usar este prompt?

Una vez creada la estructura de carpetas de tu nuevo proyecto (mediante el botón *"Crear Proyecto"*), copia este prompt maestro y envíaselo a la IA para iniciar la conversación. La IA asumirá inmediatamente todos los roles, habilidades y conocimientos de expertiz almacenados en tu carpeta `/IA/` de forma nativa.

🚀 ¿Cómo usar Agentias?

1
Elige y configura tus agentes Selecciona los roles de IA y lenguajes en este panel, y haz clic en "Crear Proyecto" para estructurar el entorno en tu disco local.
2
Carga el proyecto en tu cliente de IA Ingresa a Claude, Antigravity o Codex y abre o arrastra la carpeta del proyecto recién creado para que lea el sistema de agentes.
3
Copia, pega el prompt y comienza Copia el Prompt Maestro generado en la caja lateral, pégalo en la IA para inicializar el sistema de agentes y comienza a usar.
